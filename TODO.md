## ✅ 완료된 작업 (2026-06-27) — 세션 2

### 미래예측 차트 전면 개선
- projection을 월 단위 → **일 단위**로 전환 (더 정밀한 예측)
- 차트 Y축을 **로그 스케일**로 변경 → 과거/미래 성장 비례 표현
- 과거 실선(asset_history) + 미래 예측 점선 구분 렌더링
- 과거·예측 연결 끊김 현상 수정 (마지막 과거 포인트를 점선 시작점에 포함)
- 마일스톤 step 동적 설정 (목표 금액에 따라 1억/5억/10억/50억 단위, 최대 4개)

### 연도별 예상자산 추이 정확도 개선
- 예상 자산을 연초 기준 → **12월 말 기준** 월 단위 계산으로 변경
- 단기 목표(3년 미만) 예외처리 → 전체 FIRE 달성률만 표시
- `yearlyRows`를 projection 의존에서 독립 계산으로 변경

### FIRE 목표 달성 카드 개선
- 예상 자산: 연도 말 값 → **목표 달성 월의 자산**(목표치 근접)으로 변경
- 날짜 표시: "2027년 예상" → **"2027년 9월 예상"** (월 단위)

### asset_history 테이블 연동
